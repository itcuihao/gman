# Gman

Linux command assistant for Go.

## Getting Started

```
git clone https://github.com/itcuihao/gman.git gman

cd gman

make
```

Alternative Install Options

```
git clone https://github.com/itcuihao/gman.git gman

cd $GOPATH/gman

go install -v ./...
```
