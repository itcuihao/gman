# Gman

![.](./image/man.png)

Linux command assistant for Go.

## Getting Started

```
git clone https://github.com/itcuihao/gman.git $GOPATH/src/gman

cd gman

make
```

Alternative Install Options

```
git clone https://github.com/itcuihao/gman.git $GOPATH/src/gman

cd $GOPATH/gman

go install -v ./...
```
